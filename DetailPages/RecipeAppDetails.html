FoodRecipe
APP

Foodrecipe is a recipe app. Spoonacular API is used in this application. Recipes can be added to favourites. Recipes can be listed or added to favourites. The overview section contains the details of the recipes and the desired food types can be listed. The foodJoke section has humorous text and changes every time it is clicked.
